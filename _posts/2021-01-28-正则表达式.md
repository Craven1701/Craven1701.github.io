 `$`  结尾匹配

`^`  开头匹配

`[^]` 反向选择

```bash
[^abc] 任意不是abc的字母
```

`^$`  空白行

`.`  一定有一个任意字符

`*`  重复前一个字符0~无穷次

```
o* 所有信息
oo* 含有一个o及以上的信息
ooo* 含有两个o及以上的信息

^a.*a$ 以a开头a结尾的 任意信息
```

`.*` 任意字符

`\{\}` 范围限定

```bash
go\{2,\}g # o重复2次以上
go\{2,5\}g # o重复2~5次
```

'^A-Za-z0-9\u4e00-\u9fa5', ''
