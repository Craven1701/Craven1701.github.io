#### 匹配双引号之间的内容并清除

```bash
#正则表达式
(?<=").*?(?=") #向前匹配与向后匹配
".*?" #?表示非贪婪匹配(lazy)
#上述两个结果是有差别的，自己去regex101试一试就知道了

#全局搜索与替换
%s/".\{-}"/""/g #在vim里用这个 vim似乎识别不了？表示的非贪婪匹配 \{-} = *? 
#测试文本
0 "Part 1" 1 open
1 "Part 1A" 2 "[2 /XYZ 200 400 null]"
1 "Part 1B" 3
```

#### 使用\\(\\)和&来重现元字符匹配的串

```bash
#&会重现[parten]的部分
%s/ [0-9]/"&/g  

#适用于要还原多个模式匹配
%s/\(".*\)\ \(.*"\)/\1:\2/g 
```

#### 连续空格改为一个空格

```bash
:%s/\s\+/,/g

\+ # +表示重复多次 \转义符号
```

#### vim中删除特殊字符

```bash
sed -i ‘s/^M//g' filename
#注意：^M的输入方式是 Ctrl + v ，然后Ctrl + M 在脚本里也是这样写的
```
